# 메시지 접기 (SillyTavern 확장)

채팅 메시지 옆에 접기/펼치기 버튼을 추가합니다. `<details>`처럼 내용을 화면에서만 숨기며,
**실제 메시지 텍스트나 채팅 데이터(chat.jsonl), AI에게 전달되는 컨텍스트에는 전혀 영향을 주지 않습니다.**
접힘 상태는 브라우저 localStorage에만 저장됩니다(다른 기기/브라우저에는 동기화되지 않음).

## 설치 방법

1. 이 폴더(`mes-collapse`) 전체를 SillyTavern 설치 경로 아래 확장 폴더로 복사합니다.
   - 최신 버전: `SillyTavern/data/<사용자핸들>/extensions/mes-collapse/`
   - 구버전: `SillyTavern/public/scripts/extensions/third-party/mes-collapse/`
   - 어떤 경로인지 모르겠으면 기존 확장(예: `public/scripts/extensions/` 하위)을 찾아보고 같은 레벨에 넣으면 됩니다.
2. 폴더 안에는 `manifest.json`, `index.js`, `style.css` 세 파일이 있어야 합니다.
3. SillyTavern 페이지를 새로고침(F5)합니다.
4. 좌측 또는 상단의 확장(퍼즐 조각 아이콘) 패널을 열어 "메시지 접기"가 목록에 보이는지 확인하고, 꺼져 있다면 활성화합니다.

## 사용 방법

- 각 채팅 메시지의 버튼 줄(또는 "..." 더보기 메뉴)에 접기 아이콘이 추가됩니다.
- 클릭하면 해당 메시지 본문이 숨겨지고 "📄 내용이 접혀 있습니다" 표시줄로 바뀝니다.
- 표시줄이나 버튼을 다시 클릭하면 원래대로 펼쳐집니다.
- 새로고침 후에도 같은 채팅이라면 접은 상태가 유지됩니다(localStorage 기준).

## 참고

- ST 버전에 따라 메시지 버튼 영역의 클래스명이 다를 수 있어, 버튼이 안 보이면 브라우저 콘솔(F12)에 에러가 있는지 확인해 주세요.
- 완전히 안전하게: 이 확장은 `.mes_text`의 `display` 속성만 토글할 뿐, 메시지 데이터 자체를 수정하거나 서버로 전송하는 어떤 요청도 하지 않습니다.
